== Description ==

Choya AI is a powerful WordPress plugin that adds an intelligent AI chat assistant to your website. It integrates with OpenRouter to provide advanced AI conversations using models like Claude, GPT-4, and more.

= Free Version Features =
- Beautiful chat widget with customizable positioning
- Integration with OpenRouter API
- Support for multiple AI models (Claude, GPT-4, Gemini, etc.)
- Shortcode support for page builders
- Basic conversation history
- Responsive design that works on all devices
- Easy setup and configuration

= Pro Version Features =
- Advanced analytics and conversation tracking
- Smart triggers and behavioral automation
- Multi-language support with real-time translation
- E-commerce integration (WooCommerce, product recommendations)
- Content enhancement (quizzes, summaries, explanations)
- Voice input and output capabilities
- Lead generation and qualification
- Team collaboration features
- White label solution
- API access for developers
- Priority support

= Usage =
After installation, go to Settings > Choya AI to configure your OpenRouter API key and customize the chat widget appearance and behavior.

= Shortcodes =
- [choya_ai_chat] - Main chat widget
- [choya_ai_button] - Simple chat button
- [choya_ai_widget] - Sidebar widget

== Installation ==

= Minimum Requirements =
- WordPress 5.8 or greater
- PHP version 7.4 or greater
- OpenRouter API key (free to get)

= Automatic installation =
1. Go to the 'Add New' plugins screen in your WordPress Dashboard
2. Click the 'Upload Plugin' button
3. Upload the choya-ai.zip file
4. Activate the plugin
5. Go to Settings > Choya AI to configure

= Manual installation =
1. Upload the extracted choya-ai folder to the wp-content/plugins/ directory
2. Activate the plugin from the 'Plugins' screen in your WordPress Dashboard
3. Go to Settings > Choya AI to configure

== Frequently Asked Questions ==

= How do I get an OpenRouter API key? =
Visit https://openrouter.ai/keys and sign up for free. You'll get API access with generous free tier limits.

= Which AI models are supported? =
We support all models available on OpenRouter including:
- Anthropic Claude 3 (Sonnet, Haiku, Opus)
- OpenAI GPT-4o and GPT-4
- Google Gemini Pro
- And many more

= Is there a Pro version available? =
Yes! The Pro version includes advanced features like analytics, smart triggers, multi-language support, and more. Visit https://choya.ai/pricing for details.

= Can I use this with page builders? =
Yes! We provide shortcodes that work with Elementor, Gutenberg, WPBakery, and all other page builders.

= Is my data secure? =
Yes! We don't store your conversations unless you enable analytics. All API communication is encrypted and secure.
